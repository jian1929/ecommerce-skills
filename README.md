# E-Commerce Skills for OpenClaw

A collection of skills for e-commerce businesses to automate with OpenClaw AI agents.

## Skills

### 1. E-Commerce Competitive Monitor
Monitor competitor prices, products, and reviews for e-commerce businesses.

**Install:**
```
npx clawhub@latest install ecommerce-competitive-monitor
```

### 2. E-Commerce Social Poster
Generate and schedule social media content for e-commerce brands.

**Install:**
```
npx clawhub@latest install ecommerce-social-poster
```

### 3. E-Commerce Analytics Dashboard
Analyze e-commerce metrics and create actionable dashboards.

**Install:**
```
npx clawhub@latest install ecommerce-analytics-dashboard
```
